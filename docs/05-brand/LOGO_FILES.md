## 🎨 Logo Usage Guidelines

### **Minimum Size**

| Format | Minimum Size |
|--------|--------------|
| Digital (Screen) | 120px width |
| Print (Physical) | 25mm width |
| Favicon | 32x32px (icon only) |
| Social Avatar | 400x400px (icon only) |

---

### **Clear Space**

```
Minimum clear space = height of letter "G"

   G G G G G G G
G               G
G   ⬡ GANGCLAWCITY   G
G               G
   G G G G G G G
```

---

### **Background Usage**

#### **Dark Backgrounds (Recommended)**
```
Background: #06080f, #0a0e1a, #0d1221
Logo: Full color (white + cyan)
```

#### **Light Backgrounds**
```
Background: #ffffff, #f5f5f5
Logo: Dark version (black + cyan)
```

#### **Color Backgrounds**
```
Background: Brand colors
Logo: Monochrome (white or black)
Ensure 4.5:1 contrast ratio
```

## 📐 Logo Construction

### **Grid System**

```
The GANGCLAWCITY logo is built on a geometric grid:

1. Hexagon: Perfect regular hexagon (all sides equal)
2. Angles: 120° internal angles
3. Proportions: Height = 1.732 × width (√3)
4. Text baseline: Aligned with hexagon center

This ensures visual harmony and scalability.
```

### **Color Construction**

```
Hexagon Fill: #00e5ff (Cyber Cyan)
Hexagon Stroke: #00e5ff (2px)
GANGCLAW: #e8eaf0 (Text Primary)
CITY: #00e5ff (Cyber Cyan)
```
