# GANGCLAWCITY Start Here

GANGCLAWCITY is building its own product line by absorbing proven AI-agent products into one clearer platform. This repository is the integration workspace for that effort.

## Read This First

- Do not assume the repo is already production-ready.
- Do not assume public URLs in old reports are still valid.
- Do not treat donor repositories as the final GANGCLAWCITY product.

## What Is In Scope Today

- `openclaw-office/` is the strongest dashboard foundation.
- `agent-town/` is the strongest spatial demo foundation.
- `ai-town/`, `Star-Office-UI/`, and `pixel-agents/` are donor modules that still need adaptation before they should be presented as core GANGCLAWCITY apps.
- The current landing page is the static site at repo root: `index.html`, `style.css`, and `script.js`.

## Ground Rules

- Keep documentation truthful.
- Keep production secrets out of Markdown and browser-visible config.
- Preserve upstream license obligations while removing upstream product branding from GANGCLAWCITY-facing surfaces.
- Archive obsolete status reports instead of leaving them in active folders.
