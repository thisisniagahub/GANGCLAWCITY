## Documentation Rules

- Do not describe a deployment as live unless it has been re-verified from the current repository state.
- Do not keep temporary status reports in the repository root.
- Do not duplicate setup instructions across multiple active docs.
- When a document becomes historical, move it to `docs/99-legacy/` instead of leaving it in the active tree.
