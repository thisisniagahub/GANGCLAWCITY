## 🔧 TROUBLESHOOTING:

### **Issue: Docker tak running**
```
Solution:
1. Buka Docker Desktop
2. Tunggu hijau
3. Run SETUP_AND_RUN.bat lagi
```

### **Issue: Port dah occupied**
```bash
# Check port
netstat -ano | findstr :8080

# Kill process
taskkill /F /PID <number>
```

### **Issue: Build failed**
```bash
# Clean rebuild
docker-compose down
docker-compose build --no-cache
docker-compose up -d
```
