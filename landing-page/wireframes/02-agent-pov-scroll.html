🚶 Wireframe #2: Agent POV Scroll

Concept: Scroll through the city from an agent's perspective

📖 How It Works:

  • User follows an agent character through their daily work journey
  • Each scroll section = different district
  • Parallax effects create depth (background moves slower than foreground)
  • Agent speech bubbles explain what they're doing
  • Interactive hotspots on hover (click to learn more)
  • Mobile: swipe left/right to switch districts

🎬 Animation Breakdown

Scroll-Triggered Animations

Lottie Files - Lightweight JSON-based character animations

Speech Bubbles

Popmotion - Bouncy, playful bubble animations

Page Transitions

Barba.js - Smooth page-to-page transitions

Framer Motion - Hover effects, clicks, feedback
